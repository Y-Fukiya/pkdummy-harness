# Release Notes: v0.10.2

## Date

2026-06-19

## Scope

Public-readiness release for `pkdummy-harness`, a CLI/config-driven synthetic PK workflow fixture harness. Generated data are not for clinical inference, dose selection, or regulatory model qualification.

## Changes

- Renamed public-facing package and documentation references to `pkdummy-harness`.
- Added MIT license text.
- Added contribution and security policies for public repository use.
- Added README install guidance, CI badge, sample output, and public data/license boundary notes.
- Added project metadata for PyPI-style consumers and GitHub discoverability.
- Bumped package version to `0.10.2`.

## Known Limitations

- The harness creates workflow fixtures, not clinically validated PK predictions.
- Phoenix / NONMEM / nlmixr2 licenses and production projects are not bundled.
- Site-specific ADaM/NCA/PopPK dataset conventions may require adapter edits.
- simPop is optional; missing simPop does not block core CSV fixture generation.
