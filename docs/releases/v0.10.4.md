# Release Notes: v0.10.4

## Date

2026-06-19

## Scope

Repository operations hardening release for `pkdummy-harness`. This release adds contribution intake templates, package build checks, Markdown link checks, and release distribution assets.

Generated data remain workflow fixtures only. They are not for clinical inference, dose selection, or regulatory model qualification.

## Changes

- Added GitHub Issue templates for bugs, PK data corrections, documentation, and external tool validation reports.
- Added a Pull Request template with PK governance and validation checklists.
- Added a package build workflow that builds sdist/wheel artifacts and smoke-installs the wheel.
- Added a repository-local Markdown link checker and CI workflow.
- Updated packaging metadata to SPDX-style MIT license fields.
- Added GitHub Release assets for the built wheel and source distribution.
- Configured branch protection for `main` after workflow verification.
