# Release Notes: v0.10.3

## Date

2026-06-19

## Scope

Public hardening release for `pkdummy-harness`. This release adds citation metadata, community files, lightweight GitHub Pages documentation, and dependency/security automation.

Generated data remain workflow fixtures only. They are not for clinical inference, dose selection, or regulatory model qualification.

## Changes

- Added `CITATION.cff` citation metadata.
- Added `CODE_OF_CONDUCT.md`.
- Added a lightweight GitHub Pages documentation entrypoint under `docs/`.
- Added Dependabot configuration for GitHub Actions and Python dependencies.
- Added CodeQL workflow for Python analysis.
- Added a Pages deployment workflow for the Markdown documentation.
- Added `docs/READINESS_GAPS.md` to track tasks that require independent testers, Windows machines, or licensed external tools.
- Bumped package/citation version to `0.10.3`.
